Шпаргалка >>
-------------------------------------------------------------------------------------
TIMER0:
Внешнее тактирование производится через digital Pin 4 , частотой до 1/2 частоты МК 
Аппаратные выходы с таймера: COMPA => pin 5 , COMPB => pin 6

TIMER1:
Внешнее тактирование производится через digital Pin 5 , частотой до 1/2 частоты МК 
Аппаратные выходы с таймера: COMPA => pin 9 , COMPB => pin 10

TIMER2:
Внешнее тактирование не предусмотрено, но доступны дополнительные делители 32,128
Аппаратные выходы с таймера: COMPA => pin 11 , COMPB => pin 3
-------------------------------------------------------------------------------------

Формулы для расчета предела счета таймеров :
---------------------------------------------------------------------------------------------------------------------------
В режиме прерываний по таймеру: 
unsigned int top = 16'000'000/(N*freqency)-1; где N - делитель таймера , freqency - частота прерываний.

В режиме генерации ШИМ:
FAST PWM >>
частота ШИМ: freq = 16'000'000/N*(1+top); >> где top - предел счета и глубина шим, например глубина 10 бит >>> top = 1023
unsigned int top = 16'000'000/(N*freqency)-1; где N - делитель таймера , freqency - частота ШИМ.

PHASECORRECT PWM >>>
частота ШИМ: freq = 8'000'000/N*top; >> где N - делитель таймера, top - предел счета и глубина шим
unsigned int top = 8'000'000/N*freqency; где N - делитель таймера , freqency - частота ШИМ.

В режиме генерации меандра:
частота меандра: freq = 8'000'000/N*(1+top); >> где N - делитель таймера, top - предел счета 
unsigned int top = (8'000'000/N*freqency)-1; где N - делитель таймера , freqency - частота меандра.
----------------------------------------------------------------------------------------------------------------------------
